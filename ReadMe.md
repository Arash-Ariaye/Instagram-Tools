# Instagram Basic Tools
Little php class for Instagram.\
**Note** : \
_Don't forgot to set your own Coockie in Script._

```php
$_COOKIE = '';
```
## Main Features :
* Return Account Info (Profile)
* Download Media
* Download Highlights
* Download Story
